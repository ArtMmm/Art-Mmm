Навыки для работы

HTML5
Благодаря ему строится каркас вашего будующего сайта. А данная пятая версия позволяет мне создать более SEO-оптимизированную структуру вашего продукта.
CSS 3
Каскадные таблицы стилей третьего поколения. Позволяют мне создавать любой внешний вид вашего сайта . Все ограничивается только вашей фантазией!
JavaScript
Этот язык программирования который позволяет оживить все что угодно: слайдеры, модальные окна, подсказки, вкладки, получение данных от сервера и многое другое.
Prepros
Это маленькая но достаточная для задач верстальщика программа- сборщик которая компилирует препроцессоры Sass, Less, Stylus, Jade, Kit и другие.
WordPress
CMS - Cистема управления содержимым вашего сайта. Которая позволит вам редактировать весь ваш сайт прямиком из админ панели. Лёгкая в освоении.
Figma
Онлайн-сервис для разработки интерфейсов и прототипирования. Именно в нём сейчас очень удобно создавать дизайн вашего будующего сайта.
